# 신고하기 팝업창 소스 위치

## 핵심 파일 위치

### 1. 메인 신고 폼
**파일**: `bbs/report_form.php`
- 신고 폼의 핵심 HTML/PHP 코드
- "게시글 신고하기" 텍스트 (145번 줄)
- "신고 사유를 선택해주세요" 라벨 (153번 줄)
- 이모지 버튼들 (129-135번 줄):
  - 📨 스팸
  - 🤬 욕설
  - 🔞 음란물
  - 🐞 허위정보
  - ➕ 기타
- "취소", "신고하기" 버튼 (181-182번 줄)
- 폼 유효성 검증 및 AJAX 제출

### 2. 데스크톱 뷰 스킨
**파일**: `skin/board/report_basic/view.skin.php` (1-607번 줄)
- 모달 컨테이너 정의 (336-341번 줄)
- JavaScript 함수들 (343-523번 줄):
  - `handleReportClick()` - 신고 버튼 클릭 처리
  - `openReportForm()` - 모달 열기/폼 로딩
  - `validateReportForm()` - 폼 유효성 검증
  - `closeReportModal()` - 모달 닫기

### 3. 모바일 뷰 스킨
**파일**: `mobile/skin/board/report_basic/view.skin.php` (1-485번 줄)
- 모바일 반응형 버전 (동일한 기능)
- 신고 버튼 통합 (85번 줄)
- 모달 마크업 (297-304번 줄)
- JavaScript 제어 함수들 (306-484번 줄)

### 4. CSS 스타일링
**파일**: `css/report.css` (1-431번 줄)
- 모달 외관 스타일 (222-276번 줄)
- 이모지 아이콘 그리드 레이아웃 (291-346번 줄)
- 선택 상태 스타일 - 빨간색 하이라이트 #e74c3c (311-318번 줄)
- 버튼 스타일링 (361-397번 줄)
- 모바일 반응형 (408-417번 줄)

### 5. 설정 라이브러리
**파일**: `adm/boardreport.lib.php`
- 데이터베이스 스키마 및 설정 관리
- 3가지 아이콘 모드 지원: emoji, fa4 (FontAwesome 4), fa5 (FontAwesome 5)
- 기본 신고 사유 설정 (99번 줄)

## 주요 기능

1. **모달 구조**: 고정 위치 오버레이와 중앙 정렬 흰색 박스
2. **아이콘 모드**: 이모지, FontAwesome 4, FontAwesome 5 아이콘 지원
3. **폼 요소**: 신고 사유 선택용 라디오 버튼, 추가 의견 텍스트 영역
4. **유효성 검증**: 제출 전 클라이언트 측 유효성 검증
5. **AJAX 제출**: 페이지 새로고침 없는 폼 제출
6. **반응형 디자인**: 데스크톱 및 모바일 기기 지원
7. **접근성**: 키보드 지원 (ESC로 닫기), 포커스 관리
8. **관리자 기능**: 신고 배지 및 과도한 신고 시 콘텐츠 숨김

## iOS 작업 관련 참고사항

- PHP 기반 Gnuboard 프레임워크 사용
- 바닐라 JavaScript 및 CSS로 구현
- 모바일 스킨이 별도로 존재하여 iOS 웹뷰 최적화 가능
- AJAX를 통한 비동기 처리로 네이티브 앱과 통신 가능
